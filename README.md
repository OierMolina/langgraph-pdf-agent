# Agente LangGraph PDF (DEV-107)

Este proyecto contiene un agente basado en LangGraph capaz de generar documentos PDF con estilos personalizados (HTML/CSS a PDF) a partir de instrucciones en lenguaje natural.

## Configuración
1. Crear entorno virtual e instalar dependencias: `pip install -r requirements.txt`
2. Copiar `.env.example` a `.env` y configurar las claves.
