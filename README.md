# Agente LangGraph PDF (DEV-107)

Este proyecto es la plantilla base para un agente LangGraph orientado a la creacion de documentos PDF a partir de instrucciones en lenguaje natural, utilizando Groq como proveedor de modelo y `xhtml2pdf` para la generacion del archivo final.

## Configuracion

1. Instalar las dependencias del proyecto:

```bash
pip install -r requirements.txt
```

2. Renombrar `.env.example` a `.env` y completar la clave necesaria para el entorno.
